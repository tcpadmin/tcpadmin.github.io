node环境搭建

发表于 2018-04-02 | 分类于 编程语言

node的安装比较简单，只是需要设置一下环境变量NODE_PATH和npm的配置。否则require包时可能找不到。

安装

下载地址：http://nodejs.cn/

linux解压要用到xz解压软件，可能需要单独安装。

然后将/usr/local/node/bin添加到系统变量PATH

node不能创建软连接到PATH包含的目录下，因为npm模块的全局安装后要在node/bin中创建链接

配置

获取npm的prefix

npm在全局安装时会安装到 <prefix>/lib/node_modules中，因此要将NODE_PATH设置到这个目录，这样在node控制台和js脚本中使用require时才能找到对应包。

另外npm还可以设置缓存路径
