go限制请求次数——令牌池

发表于 2018-04-07 | 分类于 编程语言

抓数据的网站限定1秒只能有10次请求，因此设计了一个令牌管理机制来控制请求数量。

设计思路如下：

  • 发请求前需要先获取令牌
  • 限定某时间段内的发放的令牌数量
  • 任务执行完成后不能归还令牌，只能使用定时器不断重置令牌
  • 如果当前goroutine数量过多时也不重置令牌
